# Tkinter (Python 2)

1. Open **Python 2 (IDLE)** from the main menu

1. Click **File** -> **New File**

1. Enter the following code:

    ```python
    from Tkinter import *
    root = Tk()
    root.geometry("150x50")
    Label(text="Hello World").pack()
    root.mainloop()
    ```

1. Save the file as `tkinter2.py`

1. Click **Run** -> **Run Module**
